class r_treeの使い方
tree=r_tree(n次元な値のリスト)で宣言できます
n次元な値のリストというのは要素数nのリストのリストです
3次元な値のリストの例は[[1,2,3],[3,2,4],[5,1,2],[1,4,2]]です
各オペレーションは
条件を満たす要素数       count(条件)
条件を満たす要素のリスト search(条件)
でできます。
引数の(条件)は2*nの二次元リストである必要があります
たとえば
第一次元は1以上3以下,第二次元は4以上5以下,第三次元は0以上10以下の場合
[[1,3], [4,5], [0,10]]
を引数に与えれば良いです。
